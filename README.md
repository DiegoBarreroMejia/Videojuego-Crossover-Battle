# Videojuego Crossover Battle

# Descripción
Crossover Battle es un juego 2D desarrollado en Godot 4.5.1 como proyecto académico.

# Cómo descargar el juego

1. Ir a la pestaña **Releases** del repositorio.
2. En la sección **Assets** en la parte de abajo, hacer clic en: `Crossover Battle_1.exe`
